# Quickstart for Human Buyers

Pay for APIs and services using your wallet. No gas fees for you!

## Installation

```bash
bun add @armory/client-viem
```

## Pay with Your Wallet

```typescript
import { armoryPay } from '@armory/client-viem';

// Use your wallet account
const result = await armoryPay(
  { account: yourWalletAccount },
  'https://api.example.com/data',
  'base',   // network
  'usdc'    // token
);

if (result.success) {
  console.log(result.data); // Your API data
} else {
  console.error(result.code, result.message);
}
```

## What's Happening

When you pay via Armory:

1. **You create an authorization** - Sign a message off-chain (no gas!)
2. **Server verifies** - They check your signature is valid
3. **Server pays gas** - They settle the payment on-chain
4. **You get access** - Receive the API response
