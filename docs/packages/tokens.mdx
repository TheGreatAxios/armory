# @armory/tokens

Pre-configured token objects for USDC, USDT, DAI, and more across major chains.

## Installation

```bash
bun add @armory/tokens
```

## Usage

```typescript
import { TOKENS } from '@armory/tokens';

// Use with any package
import { acceptPaymentsViaArmory } from '@armory/middleware';

app.use(acceptPaymentsViaArmory({
  payTo: '0x...',
  amount: '1.0',
  token: TOKENS.USDC_BASE,
}));
```

## Available Tokens

```typescript
TOKENS.USDC_BASE        // USDC on Base
TOKENS.USDC_ETHEREUM    // USDC on Ethereum
TOKENS.USDC_POLYGON     // USDC on Polygon

TOKENS.USDT_BASE        // USDT on Base
TOKENS.USDT_ETHEREUM    // USDT on Ethereum

TOKENS.DAI_BASE         // DAI on Base
TOKENS.DAI_ETHEREUM     // DAI on Ethereum

TOKENS.WBTC_BASE        // WBTC on Base
TOKENS.WBTC_ETHEREUM    // WBTC on Ethereum
```

## Query Tokens

```typescript
import {
  getToken,
  getTokensBySymbol,
  getTokensByChain,
} from '@armory/tokens';

// By chain and address
const token = getToken(8453, '0x8335...');

// By symbol
const allUsdc = getTokensBySymbol('USDC');

// By chain
const baseTokens = getTokensByChain(8453);
```
