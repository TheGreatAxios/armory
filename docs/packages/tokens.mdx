Pre-configured token objects for USDC, USDT, DAI, WBTC, and more.

## Installation

```bash
bun add @armory-sh/tokens
```

## Usage

```typescript
import { USDC_BASE } from '@armory-sh/tokens';

import { paymentMiddleware } from '@armory-sh/middleware/bun';

const app = Bun.serve({
  fetch: paymentMiddleware({
    payTo: '0x...',
    amount: '2000000', // 2 USDC
    token: USDC_BASE,
  }),
});
```

## Available Tokens

### Base Mainnet

```typescript
import {
  USDC_BASE,        // USDC
  EURC_BASE,        // EURC
} from '@armory-sh/tokens';
```

### Base Sepolia

```typescript
import {
  USDC_BASE_SEPOLIA,
} from '@armory-sh/tokens';
```

## Token Object

Each token export contains:

```typescript
{
  symbol: string;      // e.g., "USDC"
  name: string;        // e.g., "USD Coin"
  version: string;     // Token version
  contractAddress: `0x${string}`;
  chainId: number;
  decimals: number;
}
```

## Query Functions

```typescript
import {
  getToken,
  getTokensBySymbol,
  getTokensByChain,
  getUSDCTokens,
  getAllTokens,
} from '@armory-sh/tokens';

// By chain and address
const token = getToken(8453, '0x8335...');

// By symbol
const allUsdc = getTokensBySymbol('USDC');
// or: const allUsdc = getUSDCTokens();

// By chain
const baseTokens = getTokensByChain(8453);

// All tokens
const all = getAllTokens();
```
