Convenience package that re-exports all framework-specific middleware.

## Installation

```bash
bun add @armory-sh/middleware
```

## What This Package Does

This is a **barrel package** â€” it simply re-exports everything from the framework-specific middleware packages. You can use it to import all middleware from a single package.

For detailed documentation, see:

| Package | Framework |

## Quick Import

```typescript
// Import everything from one place
import {
  createBunMiddleware,
  paymentMiddleware,
  acceptPaymentsViaArmory,
} from '@armory-sh/middleware';
```

## Tips

<Note>
For production apps, consider importing directly from the framework-specific package. This reduces bundle size and makes dependencies clearer.
</Note>

<Tip>
Use this barrel package for quick prototyping or when building a multi-framework project.
</Tip>
